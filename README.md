# ✨ MagicMap

O **MagicMap** é um guia interativo de rotas projetado para transformar sonhos em realidade. Ele permite que os usuários escolham entre os principais parques temáticos de Orlando e obtenham rotas precisas via Google Maps utilizando a geolocalização do dispositivo.

Este projeto foi desenvolvido como uma **PWA (Progressive Web App)**, o que significa que ele pode ser instalado em dispositivos móveis e funciona offline graças ao uso de Service Workers.

---

## 🚀 Funcionalidades

* **Interface Mágica**: Design imersivo com gradientes vibrantes, estrelas animadas e temática inspirada no universo Disney.
* **Seleção de Destinos**: Menu intuitivo para escolher entre Magic Kingdom, Epcot, Hollywood Studios e Animal Kingdom.
* **Integração com Google Maps**: Cálculo de rota em tempo real a partir da localização atual do usuário.
* **Modo Offline**: Suporte a PWA com manifest e cache para acesso rápido.
* **Design Responsivo**: Otimizado para visualização perfeita em dispositivos móveis (Mobile First).

---

## 🛠️ Tecnologias Utilizadas

Como desenvolvedora interessada em **Java, JavaScript, HTML, CSS e SQL**, utilizei as seguintes ferramentas neste projeto:

* **HTML5**: Estrutura semântica das telas.
* **CSS3**: Estilização avançada com `flexbox`, `linear-gradients` e animações de estrelas (`@keyframes`).
* **JavaScript (ES6)**: Lógica de transição de telas, manipulação de DOM e API de Geolocalização.
* **PWA**: Service Workers (`sw.js`) e Web App Manifest (`manifest.json`) para instalação e cache.

> **Nota**: Os botões são arredondados (estilo pílula) e em negrito para garantir uma experiência de toque amigável no mobile.

---

## 🔧 Como Executar

1. Clone este repositório.
2. Certifique-se de que as imagens `imageapp.jpg` e `parques.jpg` estejam na pasta raiz.
3. Abra o arquivo `index.html` em qualquer navegador moderno.
4. Para testar a geolocalização, certifique-se de permitir o acesso à localização quando solicitado.
